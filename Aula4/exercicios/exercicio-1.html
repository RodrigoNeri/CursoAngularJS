Curso AngularJS - Aula 4 Exercício 1

Desta vez, isto aqui é um arquivo que vai dizer o que vocês devem fazer.

  •   Vocês devem iniciar uma aplicação AngularJS do zero;
  •   Deve ser buscado qualquer perfil que possa ser buscado utilizando um API. Recomendo a API do Github, mas pode ser utilizada qualuqer uma. (https://api.github.com/users/mhevery)
  •   Devem ser buscados 3 perfis diferentes e exibidos;
  •   Para buscar o conteúdo na API, utilize o que você achar melhor do que vimos até agora;
  •   Deve ser criada uma diretiva para exibir o perfil do usuário; Recomendo a utilização de algo pronto para CSS como o Bootstrap.

Referências que podem ser úteis:

  • AngularJS CDN: https://code.angularjs.org/1.3.15/angular.min.js
  • Bootstrap CDN: https://maxcdn.bootstrapcdn.com/bootstrap/3.3.4/css/bootstrap.min.css
  • Exemplo de diretivas visto na aula passada;
  • Github com todos os códigos que já vimos até agora: https://github.com/luisdalmolin/CursoAngularJS
